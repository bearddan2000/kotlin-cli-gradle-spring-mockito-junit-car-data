# kotlin-cli-gradle-spring-mockito-junit-car-data

## Description
A POC for spring app using junit,
mockito framework.

## Tech stack
- kotlin
- gradle
  - spring
  - junit
  - mockito

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
